# Homelab Docs

Private, sanitized reference docs for the Auala Ohana homelab.

## Purpose

This repo exists to preserve architecture, operational decisions, backup/recovery notes, and service layout without storing live secrets.

## Principles

- No tokens, passwords, API keys, or private credentials
- Favor roles and patterns over sensitive implementation detail
- Capture enough context to rebuild understanding after outages or time away
- Prefer plain operational truth over aspirational diagrams
